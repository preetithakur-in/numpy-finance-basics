# NumPy Finance Basics
This project demonstrates basic financial analysis using NumPy.

## Objective
To understand how numerical computing can be applied in finance, including:
- Daily returns calculation
- Volatility (standard deviation of returns)
- Data filtering
- Cumulative returns

## Technologies Used
- Python
- NumPy

## Key Concepts

### 1. Daily Returns
Calculated using:
(price_today - price_yesterday) / price_yesterday

### 2. Volatility
Standard deviation of daily returns, used to measure risk.

### 3. Data Filtering
Using boolean indexing to extract values based on conditions.

### 4. Cumulative Returns
Calculated using:
cumprod(1 + returns)

This project helped in understanding how NumPy can be applied to financial data analysis and forms the foundation for further work using Pandas and real-world datasets.
